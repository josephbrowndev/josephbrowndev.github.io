Project Overview

Limitless Runner is a published Steam game that showcases advanced procedural generation and animation design. The game features a sophisticated tile-spawning algorithm that continuously generates platforms while efficiently culling off-screen tiles to maintain optimal performance. Players control a charming mouse protagonist with 12 distinct animation states, collecting berries for points while dodging or destroying bee enemies. The strategic placement of obstacles forces lateral movement decisions, while cheese power-ups provide game-changing abilities like slow-motion, invincibility, and double-jump mechanics.

Core Innovations

Procedural Level Algorithm

Infinite variety through 5 platform types: flat, ascending, descending, swing left, and swing right - all seamlessly generated

Advanced Animation Design

Fluid mouse character animations with dynamic grappling hook mechanics that appear and disappear based on context

Technical Implementation

Procedural Level Generation Algorithm

  • Continuous Tile Spawning: Algorithm generates platforms ahead of player view, ensuring seamless infinite gameplay
  • Intelligent Culling System: Tiles behind player are automatically deleted, maintaining constant memory footprint regardless of session length
  • Platform Pattern Variety: Five distinct types - flat segments for speed, ascending/descending for verticality, swing patterns for dynamic movement
  • Berry Placement Logic: Strategic collectible spawning that rewards skill while maintaining achievable paths
  • Difficulty Curve Integration: Algorithm adjusts spacing, obstacle density, and platform complexity based on player progress
  • Seamless Transitions: Smart blending ensures no impossible gaps or jarring difficulty spikes between segments

Performance Optimization

  • Object Pooling System: Reusable tile, enemy, and particle pools eliminate garbage collection stutters
  • Efficient Culling Pipeline: Off-screen elements immediately deactivated to minimize draw calls and physics calculations
  • Texture Atlasing: Combined sprite sheets reduce texture swaps and improve rendering batch efficiency
  • LOD Implementation: Background and distant elements use simplified assets based on camera distance
  • Frame Rate Stability: Consistent 60+ FPS maintained across various hardware configurations
  • Memory Management: Constant memory footprint regardless of play session length through aggressive cleanup
